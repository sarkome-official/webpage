# Sarcoma.dev Frontend

Astro-based frontend for Sarcoma.dev research platform.

## Setup

```bash
npm install
npm run dev
```

## Adding shadcn/ui Components

```bash
npx shadcn@latest add [component-name]
```

Components will be installed to `src/components/ui/`.

## Build

```bash
npm run build
npm run preview
```
